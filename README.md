## About The Project

Spotify clone is a educational proyect which tries to reproduce the famous Spotify app with React

## Getting Started

Install the dependencies:

```sh
$ npm install
// or
$ yarn
```

Run in dev mode:

```sh
$ npm run dev
// or
$ yarn dev
```

### Built With

- JavaScript
- [Vite](https://vitejs.dev)
- [React](https://reactjs.org/)
